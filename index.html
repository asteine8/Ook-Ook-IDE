Ook Ook IDE

This IDE runs Ook!, an esoteric programming language. Input and output are interpreted as ASCII characters (string.charCodeAt(0)). Characters not in the Ook! command set will be interpreted as comments. This interpreter adds an extension to Ook! using the Random Brainfuck specifications where Ook? Ook? will write a random 1 or 0 to the current cell.
Also note that if text is in the console input buffer, the left-most character will be consumed in the case of an input command. If more text needs to be added, type the text then press the enter key to set the buffer.
